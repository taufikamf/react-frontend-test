## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

_All you need to do to run this app in your local._

1. Clone the repo
   ```sh
   git clone https://github.com/taufikamf/react-frontend-test
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run your project
   ```sh
   npm start
   ```
